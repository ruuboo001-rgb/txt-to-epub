# TXT → EPUB Studio v2.6 (Streamlit 웹버전)

## v2.6 변경 사항

- `회차 위 반복 작품제목 제거` 옵션을 추가했습니다.
  - 기본값은 꺼짐입니다.
  - `소설제목 → 빈 줄 → 1화`, `소설제목 → 빈 줄 → 2화`처럼 회차 바로 위에 작품 제목이 반복될 때만 켜세요.
  - 작품 제목과 정확히 같은 줄 바로 아래에 회차 제목이 있을 때만 제거합니다.
  - 본문 중간에 우연히 작품 제목이 나온 경우는 아래가 회차 제목이 아니면 건드리지 않습니다.
- `제거할 반복 머리말` 입력칸을 추가했습니다.
  - 비워두면 작품 제목 입력값을 기준으로 처리합니다.
  - 작품 제목과 다른 문구가 반복될 때만 따로 적으면 됩니다.

## 기존 주요 기능

- TXT 업로드 → EPUB 생성
- 표지/표제지/꾸민 목차 생성 선택
- 회차 자동 감지 및 직접 정규식/예시 패턴 생성
- 잘못 감지된 회차 제외 후 본문 복구
- 원본 목차 자동 제거
- 긴 빈 줄을 `<br>` 공백으로 보존하는 선택 기능
- 여러 줄 점 전환 표시 중앙 정렬
- 본문 자동 들여쓰기 ON/OFF
- 본문 글꼴 지정 ON/OFF 및 외부 폰트 동봉

## Streamlit 배포

GitHub 저장소에 압축을 푼 파일을 덮어올린 뒤 Streamlit Community Cloud에서 `app.py`를 메인 파일로 배포하면 됩니다.


## v2.6 변경사항

- 웹버전에서도 TXT 파일을 올리면 파일명에서 확장자를 뺀 이름이 `제목` 칸에 자동으로 입력됩니다.
- `TXT 파일명으로 제목 자동 입력` 옵션을 추가했습니다. 기본값은 켜짐입니다.
- 직접 제목을 쓰고 싶으면 옵션을 끄거나 제목 칸을 수정하면 됩니다.
- 파일명이 바뀌면 이전 제목이 비어 있거나 자동 입력된 제목이었을 때만 새 파일명으로 다시 채웁니다.
